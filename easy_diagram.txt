=====================================

ДЛЯ РИСОВАНИЯ ВРУЧНУЮ:

1. Нарисуйте 5 прямоугольников друг под другом
2. В каждом прямоугольнике напишите:
   - Пользователь
   - Фронтенд (HTML+JS+CSS)
   - Flask приложение
   - Бизнес-логика
   - База данных
3. Нарисуйте стрелки между блоками (сверху вниз)
4. Раскрасьте разными цветами

=====================================

ДЛЯ ПРЕЗЕНТАЦИИ:

Можно использовать простые фигуры в PowerPoint/Google Slides:
- Прямоугольники для блоков
- Стрелки для связей
- Текст внутри блоков
- Цветовое кодирование
